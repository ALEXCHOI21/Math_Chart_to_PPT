## 🔧 2. 개선 계획 (솔루션 설계)

### 2.1 Root Cause: 왜 웹사이트가 작동하지 않는가?

**3가지 이유**:

1. **배포 아키텍처 오류**
   ```
   GitHub Pages ➜ 정적 호스팅만 지원
                 ➜ 서버 실행 불가
                 ➜ FastAPI 백엔드 배포 불가
   ```
   **해결**: 별도 클라우드 서버 (Render, Railway, Vercel) 필요

2. **API 엔드포인트 연결 끊김**
   ```javascript
   script.js: fetch('http://localhost:8000/upload')
   ❌ 프론트엔드는 GitHub Pages (https://alexchoi21.github.io)
   ❌ 백엔드는 로컬 머신 (localhost:8000)
   ➜ CORS + 404 에러 발생
   ```
   **해결**: 백엔드 클라우드 배포 후 API_BASE_URL 수정

3. **환경 변수 관리 부재**
   ```
   현재: API 주소 하드코딩
   개선: .env 파일로 환경별 설정 분리
        - 로컬: http://localhost:8000
        - 프로덕션: https://api.choigpt.com
   ```

### 2.2 단계별 개선 로드맵

#### **Phase 1: 긴급 수정 (48시간) - P0 우선순위**

**목표**: 웹사이트 정상 작동

**작업 내용**:

1️⃣ **백엔드 클라우드 배포**
   - 플랫폼 선택: **Render** (추천)
     ```
     장점: FastAPI 자동 감지 + 무료 티어
     비용: 월 $7 (Pro)
     시간: 30분
     ```
   - 배포 단계:
     ```bash
     1. Render.com 가입
     2. GitHub 연결 (Math_Chart_to_PPT 리포지토리)
     3. New Service → Web Service → GitHub 선택
     4. Root Directory: backend
     5. Build Command: pip install -r requirements.txt
     6. Start Command: uvicorn main:app --host 0.0.0.0
     ```
   - 결과: `https://math-chart-api.onrender.com` (예시)

2️⃣ **프론트엔드 API 엔드포인트 수정**
   ```javascript
   // script.js 수정
   
   // Before (❌ 하드코딩)
   const API_URL = 'http://localhost:8000';
   
   // After (✅ 환경 변수)
   const API_URL = process.env.REACT_APP_API_URL || 
                   'https://math-chart-api.onrender.com';
   ```

3️⃣ **CORS 설정 수정**
   ```python
   # backend/main.py
   from fastapi.middleware.cors import CORSMiddleware
   
   app.add_middleware(
       CORSMiddleware,
       allow_origins=[
           "https://alexchoi21.github.io",
           "http://localhost:3000"
       ],
       allow_methods=["*"],
       allow_credentials=True,
   )
   ```

4️⃣ **GitHub Pages에서 환경 변수 설정**
   ```bash
   # GitHub Actions 워크플로우 생성
   # .github/workflows/deploy.yml
   
   name: Deploy Frontend
   on: [push]
   jobs:
     deploy:
       runs-on: ubuntu-latest
       steps:
         - uses: actions/checkout@v3
         - name: Deploy to GitHub Pages
           env:
             REACT_APP_API_URL: https://math-chart-api.onrender.com
           run: |
             cd frontend
             npm install
             npm run build
   ```

**산출물**: ✅ 웹사이트 정상 작동 (업로드 → 분석 → 다운로드)  
**소요 시간**: 2-3시간
